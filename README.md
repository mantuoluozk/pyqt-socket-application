# pyqt-socket-简易微信

## 使用

先运行server.py,开启服务器(python server.py) 
再运行client.py,开启客户端(python client.py), 开启多个客户端相互通信 

## 功能
* 注册，登录
* 建群，加好友
* 好友上下线通知
* 私聊，群聊
* 发送文字，图片
> 因为没有使用数据库，所以每次服务器关闭后，所有信息都会消失
